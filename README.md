

install node modules
```
npm install
```

## Folder Structure
After creation, your project should look like this:

```
react-simple-restaurant-menu/    
    node_modules/
    public/
    src/   
    	assets/
    	routes/ 
    	data/   	     
        component/
            component1/
                component.js
                component.test.js
        ...
        index.js
    .gitignore
    package.json
    README.md
```


## Run the project

```
npm start
```

## Building the project

```
npm run build
```
